# Enterprise Network Automation and Monitoring System

## Project Overview
This project demonstrates a modern approach to managing an Enterprise Network (3-Tier Architecture) using NetDevOps methodologies. It integrates network automation for configuration management and real-time monitoring.

## Architecture & Technologies
* Network Topology: 3-Tier Enterprise Network (Core - Distribution - Access) simulated on EVE-NG.
* Automation Server: Ubuntu Linux running Python (Netmiko/NAPALM) and Ansible.
* Monitoring Stack: Prometheus (SNMP Exporter) & Grafana for visualization.
* Routing Protocols: OSPF, BGP.

## Key Features
* Automated Provisioning: Push configurations (VLANs, STP, Routing) across multiple devices automatically.
* Configuration Management: Scheduled automated backups of all network equipment.
* Real-time Monitoring: Tracking critical device metrics (CPU, Memory, Interface bandwidth) via custom Grafana Dashboards.
